🍜 App Idea: กินไรดีวะ

- แอปสุ่มร้านอาหาร / เมนู สำหรับคนที่ “คิดไม่ออกว่าจะกินอะไร”

🎯 กลุ่มเป้าหมาย

- คนทำงาน / นักศึกษา

- คนอยู่คอนโด

- กลุ่มเพื่อนที่เถียงกันนานก่อนกินข้าว 😆

⭐ Core Concept

- เปิดแอป → เลือกเงื่อนไขนิดหน่อย → กดปุ่มเดียว → ได้คำตอบทันที

📱 ฟีเจอร์หลัก (MVP)
1. ปุ่มสุ่มใหญ่กลางจอ

- ปุ่ม “สุ่มให้หน่อย!”

- Animation หมุน / เขย่า / slot machine

2. ตัวกรองก่อนสุ่ม

- 💰 งบประมาณ (≤50 / 100 / 200 / 300+)

- 🍔 ประเภทอาหาร (ตามสั่ง, ชาบู, คาเฟ่, ฟาสต์ฟู้ด)

- 📍 ระยะทาง (ใกล้ 500m / 1km / 3km)

- 🕒 เวลา (เช้า / เที่ยง / เย็น / ดึก)

3. แสดงผลลัพธ์

- ชื่อร้าน + รูป

- ระยะทาง

- ราคาโดยประมาณ

- ปุ่ม “นำทาง” (Google Map)

- ปุ่ม “สุ่มใหม่”

🔥 ฟีเจอร์เสริม (Phase 2)

- ❤️ Favorite ร้านโปรด

- ❌ Blacklist ร้านที่ไม่อยากกิน

- 👥 โหมดเพื่อน (โหวตร้าน)

- 🎲 โหมด “ไม่รู้จะกินอะไรจริง ๆ” (ไม่ต้องเลือก filter)
